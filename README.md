# AeorDB Client

Sync-first native desktop client for [AeorDB](https://github.com/th317erd/aeordb). Keeps local directories in continuous sync with remote AeorDB servers — bidirectional, pull-only, or push-only.

- **Native desktop app** — Rust + Tauri, runs headed (systray) or headless
- **File browser** — browse, preview, and manage synced files
- **Sync management** — configure relationships, monitor activity, trigger manual syncs
- **Connection dashboard** — embedded remote AeorDB portal per connection
- **CLI** — full control from the terminal
